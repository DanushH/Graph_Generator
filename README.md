<h1 align="center">Graph Generator</h1>

<br/>

## Overview
- A standalone application developed with [PySimpleGUI](https://docs.pysimplegui.com/en/latest/), a GUI library for Python.
- [Matplotlib](https://matplotlib.org/stable/index.html), a plotting library for Python, is utilized to create graphs.
- [Numpy](https://numpy.org/doc/stable/), a computational library for Python, is utilized to calculate variable values.
- This application provides functionalities for calculating data points and plotting graphs for linear equations, quadractic equations, cubic equations, reciprocal equations and exponential equations.

<br/>

## Getting Started
Run your own local copy by following these steps.

### Prerequisites
- Install [Python](https://www.python.org/downloads/).

<br/>

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/DanushH/Graph_Generator.git
   ```

2. Install Dependencies:
   <br/>
   Install PySimpleGUI:
   ```sh
   pip install PySimpleGUI
   ```
   Install NumPy:
   ```sh
   pip install numpy
   ```
   Install Matplotlib:
   ```sh
   pip install matplotlib
   ```

3. Run the Application:
   ```sh
   python main.py
   ```
